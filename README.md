# 🛒 Retail Transactions Dataset

## 📌 Overview

This dataset contains retail transaction data capturing sales across different regions, categories, and sales channels. It is useful for data analysis, dashboard creation, and business insights.

---

## 📊 Dataset Details

* **Total Rows:** 20
* **Total Columns:** 11

---

## 🧾 Columns Description

| Column Name   | Description                                   |
| ------------- | --------------------------------------------- |
| TransactionID | Unique ID for each transaction                |
| Date          | Date of transaction                           |
| ProductName   | Name of the product                           |
| Category      | Product category (Electronics, Fashion, etc.) |
| Region        | Sales region (East, West, North, South)       |
| SalesChannel  | Mode of sale (Online / Offline)               |
| Quantity      | Number of units sold                          |
| UnitPrice     | Price per unit                                |
| TotalAmount   | Total transaction value                       |
| PaymentMode   | Payment method (UPI, Cash, Credit Card, etc.) |
| CustomerID    | Unique customer identifier                    |
